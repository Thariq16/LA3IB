# MVP Screens to Design

## 1. Onboarding
*   **Phone number login** (Saudi-friendly)
*   **Gender selection** (Men / Women / Mixed)
*   **Preferred sport selection** (limit to 2)
*   **City / area selection**

## 2. Home / Game Discovery
*   **List and map view** of nearby games
*   **Filters:**
    *   Sport
    *   Gender
    *   Date / time
*   **Game cards showing:**
    *   Sport icon
    *   Time & date
    *   Location
    *   Slots available
    *   Price

## 3. Create Game
*   **Simple form:**
    *   Sport
    *   Gender rule
    *   Max players
    *   Price per player
    *   Date & time
    *   Location
*   **Primary CTA:** “Create Game”

## 4. Game Details
*   **Clear game summary**
*   **Participant list**
*   **Join button**
*   **Payment required indicator**

## 5. Payment Flow
*   **Simple, trusted payment UI**
*   **Price breakdown**
*   **Confirmation screen**

## 6. My Games
*   **Upcoming games**
*   **Joined vs created games**

## 7. Notifications
*   **Game confirmed**
*   **Slot filled**
*   **Game cancelled**

## 8. Navigation
*   **Bottom navigation with:**
    *   Home
    *   Create
    *   My Games
    *   Profile
